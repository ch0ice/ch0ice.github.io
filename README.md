## Hexo Options
hexo server #启动本地服务
hexo clean #清除缓存 网页正常情况下可以忽略此条命令
hexo generate #生成静态网页
hexo generate && gulp #生成静态网页 并且对文件进行压缩
hexo deploy #开始部署

也可以一次性执行
hexo clean && hexo generate && gulp && hexo deploy
